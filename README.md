# IAMasterMind
AI project implementing a Machine Learning algorithm for the MasterMind game, allowing the algorithm to learn how to play by itself.

## Mastermind IA (Q-Learning)
A web implementation of the classic logic game Mastermind, featuring a single-player mode and an Artificial Intelligence agent capable of learning to play and solving the code on its own using Reinforcement Learning.

## Technologies Used
- **HTML5 & CSS3:** Responsive design using Flexbox and Grid.
- **JavaScript (ES6):** Game logic and AI algorithm.
- **jQuery:** DOM manipulation.
- **SweetAlert:** Stylized modal alerts.
- **Google Fonts:** VT323 typography.

## Installation and Usage

```bash
First, clone the repository:

git clone https://github.com/iZeroZack/IAMasterMind.git

Then, open the project folder and open the index.html file in your browser.
```
